# Recursion in Python

This folder contains Python programs based on recursion, starting from basic recursive problems and moving toward more advanced concepts and applications.

Recursion is a programming technique where a function calls itself in order to solve smaller instances of the same problem. It is one of the most important concepts in computer science because it helps in solving complex problems in a clean and elegant way.

In this folder, the programs cover:
- basic recursive functions
- factorial and Fibonacci series
- sum of numbers using recursion
- power calculation
- greatest common divisor
- palindrome checking
- decimal to binary conversion
- recursive searching
- Tower of Hanoi
- merge sort using recursion

These programs are useful for understanding:
- base case and recursive case
- problem decomposition
- mathematical problem solving
- divide and conquer approach
- algorithmic thinking

This folder is designed to show a step-by-step progression from beginner recursion programs to advanced recursive problem solving in Python.
