# Recursion in Python

This folder contains Python programs that demonstrate both foundational and advanced recursion concepts. The programs begin with simple recursive examples and gradually move toward divide-and-conquer algorithms, combinatorial generation, backtracking, and constraint-solving problems.

Recursion is one of the most important problem-solving techniques in computer science. In recursion, a function calls itself to solve smaller instances of the same problem until it reaches a stopping condition called the base case. This approach is widely used in mathematical computation, tree and graph traversal, searching, sorting, dynamic programming, backtracking, and many algorithmic interview problems.

This folder includes:
- basic recursion problems
- divide-and-conquer algorithms
- recursive searching and sorting
- subset and permutation generation
- backtracking problems
- constraint-based recursive solvers

The goal of this folder is not only to show how recursion works, but also to demonstrate how recursion can be applied to solve complex real-world and interview-style problems in a structured and elegant way.

Advanced topics covered in this folder:
- Quick Sort
- Recursive Binary Search
- Subset Generation
- Permutation Generation
- N-Queens Problem
- Maze Solver
- Sudoku Solver

These programs help in understanding:
- base cases and recursive calls
- stack-based problem solving
- divide-and-conquer strategy
- state-space exploration
- decision trees
- pruning and backtracking
- algorithm design for complex problems

This folder is designed to show a clear progression from beginner-level recursion to advanced recursive and backtracking-based problem solving in Python.
