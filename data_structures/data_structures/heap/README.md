# Heap in Python

This folder contains Python programs based on heaps, one of the most important specialized tree-based data structures in computer science.

A heap is a complete binary tree that satisfies the heap property. In a min-heap, the parent node is smaller than or equal to its children. In a max-heap, the parent node is greater than or equal to its children. Heaps are widely used in priority queues, scheduling systems, graph algorithms, top-k problems, and efficient data processing.

Topics covered in this folder:
- heap basics
- min-heap
- max-heap simulation
- insertion in heap
- deletion from heap
- heapify operation
- heap sort
- priority queue usage
- kth largest and kth smallest problems
- practical heap-based problem solving

This folder is designed to build a strong understanding of heap fundamentals before moving into priority queues, graph algorithms, greedy methods, and advanced interview-style problems.
