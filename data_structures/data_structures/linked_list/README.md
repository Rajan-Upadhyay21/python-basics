# Linked List in Python

This folder contains Python programs based on linked lists, one of the most important linear data structures in computer science.

A linked list stores data in nodes, where each node contains a value and a reference to the next node. Unlike arrays, linked lists do not require contiguous memory and make insertion or deletion easier at certain positions.

Topics covered in this folder:
- singly linked list
- traversal
- insertion
- deletion
- searching
- length calculation

This folder is designed to build a strong understanding of linked list fundamentals before moving into stacks, queues, trees, and more advanced data structures.
