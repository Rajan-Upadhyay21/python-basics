# Tree in Python

This folder contains Python programs based on trees, one of the most important non-linear data structures in computer science.

A tree is a hierarchical data structure made up of nodes connected by edges. It is widely used in file systems, databases, compilers, search systems, expression evaluation, and many algorithmic problems. Tree-based thinking is also very important for coding interviews and advanced problem solving.

Topics covered in this folder:
- binary tree basics
- tree traversal
- binary search tree
- insertion in BST
- searching in BST
- tree height
- counting nodes
- counting leaf nodes
- level order traversal
- basic recursive tree problems

This folder is designed to build a strong understanding of tree fundamentals before moving into heaps, graphs, recursion-heavy algorithms, and advanced tree-based problem solving.
