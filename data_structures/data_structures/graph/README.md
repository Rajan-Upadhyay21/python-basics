# Graph in Python

This folder contains Python programs based on graphs, one of the most important non-linear data structures in computer science.

A graph is a collection of vertices (nodes) and edges that connect them. Graphs are widely used in social networks, maps, routing systems, dependency management, recommendation systems, network design, and many advanced algorithmic problems.

Topics covered in this folder:
- graph representation
- adjacency list
- adjacency matrix
- breadth-first search
- depth-first search
- connected components
- cycle detection
- shortest path basics
- graph traversal problems
- practical graph-based problem solving

This folder is designed to build a strong understanding of graph fundamentals before moving into graph algorithms such as Dijkstra, topological sorting, union-find, minimum spanning tree, and advanced interview-style problems.
