# Stack in Python

This folder contains Python programs based on the stack data structure, one of the most important linear data structures in computer science.

A stack follows the LIFO principle, which means Last In, First Out. The last element inserted into the stack is the first one to be removed. Stacks are widely used in expression evaluation, function calls, recursion, undo operations, syntax checking, and many algorithmic problems.

Topics covered in this folder:
- stack implementation using list
- push operation
- pop operation
- peek operation
- checking if stack is empty
- stack size
- balanced parentheses
- reverse string using stack
- practical stack-based problems

This folder is designed to build a strong understanding of stack fundamentals before moving into queues, trees, recursion-heavy problems, and advanced algorithm design.
