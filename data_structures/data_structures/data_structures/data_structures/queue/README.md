# Queue in Python

This folder contains Python programs based on the queue data structure, one of the most important linear data structures in computer science.

A queue follows the FIFO principle, which means First In, First Out. The first element inserted into the queue is the first one to be removed. Queues are widely used in scheduling, buffering, task processing, BFS traversal, simulations, and many real-world system designs.

Topics covered in this folder:
- queue implementation using list
- enqueue operation
- dequeue operation
- front and rear access
- checking if queue is empty
- queue size
- circular queue basics
- queue using collections.deque
- practical queue-based problems

This folder is designed to build a strong understanding of queue fundamentals before moving into trees, graphs, BFS, and advanced algorithm design.
