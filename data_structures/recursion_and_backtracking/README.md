# Recursion and Backtracking in Python

This folder contains Python programs based on recursion and backtracking, two of the most important problem-solving techniques in computer science.

Recursion is a method where a function calls itself to solve smaller instances of the same problem until a base case is reached. Backtracking is an extension of recursion where we explore possible choices, undo them when they do not lead to a valid solution, and continue searching for the correct path.

These techniques are widely used in combinatorial problems, maze solving, constraint satisfaction, permutations, subsets, Sudoku, N-Queens, and many advanced interview-style problems.

Topics covered in this folder:
- factorial and Fibonacci using recursion
- recursive search and divide-and-conquer ideas
- subset generation
- permutation generation
- combination generation
- maze solving
- N-Queens problem
- Sudoku solver
- balanced parentheses generation
- backtracking-based decision trees

This folder is designed to build a strong understanding of recursive thinking and systematic exploration before moving into greedy algorithms, dynamic programming, and advanced algorithm design.
