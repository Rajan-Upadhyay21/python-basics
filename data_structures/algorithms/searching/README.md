# Searching Algorithms in Python

This folder contains Python programs based on searching algorithms, which are among the most fundamental algorithms in computer science.

Searching algorithms are used to locate an element, value, or condition inside a data structure efficiently. They are widely used in databases, applications, problem solving, interview questions, and algorithm design.

Topics covered in this folder:
- linear search
- binary search
- recursive binary search
- jump search
- interpolation search
- first and last occurrence
- search in rotated sorted array
- searching in 2D matrix
- practical searching problems

This folder is designed to build a strong understanding of searching techniques before moving into sorting, greedy methods, dynamic programming, and advanced algorithmic problem solving.
