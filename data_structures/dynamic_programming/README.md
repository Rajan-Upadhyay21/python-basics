# Dynamic Programming in Python

This folder contains Python programs based on dynamic programming, one of the most important problem-solving techniques in computer science.

Dynamic programming is used when a problem has overlapping subproblems and optimal substructure. Instead of solving the same subproblem again and again, dynamic programming stores previous results and reuses them. This makes solutions much more efficient than plain recursion for many problems.

Dynamic programming is widely used in optimization, path finding, sequence problems, decision making, resource allocation, and many high-level interview and competitive programming problems.

Topics covered in this folder:
- Fibonacci with memoization
- climbing stairs
- coin change
- house robber
- longest common subsequence
- longest increasing subsequence
- knapsack
- subset sum
- edit distance
- grid-based dynamic programming

This folder is designed to build a strong understanding of dynamic programming before moving into advanced optimization problems, graph DP, interval DP, and harder interview-style algorithmic problem solving.
