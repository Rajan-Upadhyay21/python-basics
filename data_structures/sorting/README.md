# Sorting Algorithms in Python

This folder contains Python programs based on sorting algorithms, which are among the most important algorithms in computer science and problem solving.

Sorting algorithms are used to arrange data in a particular order, usually ascending or descending. They are widely used in searching, optimization, data processing, ranking systems, scheduling, and many coding interview problems.

Topics covered in this folder:
- bubble sort
- selection sort
- insertion sort
- merge sort
- quick sort
- heap sort
- counting sort
- radix sort
- bucket sort
- practical sorting-based problems

This folder is designed to build a strong understanding of sorting techniques before moving into greedy algorithms, dynamic programming, advanced graph algorithms, and interview-style algorithmic problem solving.
