# Greedy Algorithms in Python

This folder contains Python programs based on greedy algorithms, one of the most important problem-solving techniques in computer science.

A greedy algorithm builds a solution step by step by always choosing the locally optimal option at each stage. Greedy methods are widely used in scheduling, optimization, interval problems, graph algorithms, Huffman coding, minimum cost problems, and many interview-style questions.

Topics covered in this folder:
- activity selection
- coin change greedy
- fractional knapsack
- minimum platforms
- job sequencing
- meeting room scheduling
- interval merging
- gas station circuit
- Huffman coding basics
- practical optimization problems

This folder is designed to build a strong understanding of greedy thinking before moving into dynamic programming, graph optimization, and advanced algorithm design.
