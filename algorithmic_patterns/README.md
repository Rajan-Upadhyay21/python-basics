# Algorithmic Patterns in Python

This folder contains Python programs based on algorithmic patterns, which are recurring problem-solving techniques used across data structures, algorithms, coding interviews, and competitive programming.

Algorithmic patterns help in recognizing the structure of a problem and applying the right strategy efficiently. Instead of memorizing isolated problems, learning patterns makes it easier to solve a wide range of questions with similar logic.

Topics covered in this folder:
- two pointers
- sliding window
- fast and slow pointers
- binary search pattern
- merge intervals
- prefix sum
- monotonic stack
- top k elements
- backtracking pattern
- dynamic programming pattern

This folder is designed to build a strong understanding of common problem-solving templates that can be reused across many interview-style and real-world programming problems.
