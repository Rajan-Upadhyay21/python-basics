# Graph Algorithms in Python

This folder contains Python programs based on graph algorithms, which are among the most important advanced topics in computer science and problem solving.

Graph algorithms are used to process relationships between nodes and edges. They are widely used in shortest path problems, network routing, dependency resolution, scheduling, social networks, maps, recommendation systems, and many interview-style algorithmic questions.

Topics covered in this folder:
- breadth-first search
- depth-first search
- Dijkstra's algorithm
- Bellman-Ford algorithm
- Floyd-Warshall algorithm
- topological sorting
- cycle detection
- minimum spanning tree
- union-find
- strongly connected components

This folder is designed to build a strong understanding of graph-based problem solving and optimization in Python.
