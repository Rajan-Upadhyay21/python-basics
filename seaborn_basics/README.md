# Seaborn Basics in Python

This folder contains Python programs based on Seaborn, a powerful Python library for statistical data visualization.

Seaborn is built on top of Matplotlib and provides a high-level interface for creating attractive and informative statistical graphics. It is widely used in data analysis, exploratory data analysis, machine learning projects, and visual storytelling.

Topics covered in this folder:
- line plots
- bar plots
- count plots
- histograms
- box plots
- violin plots
- scatter plots
- pair plots
- heatmaps
- distribution plots

This folder is designed to build a strong foundation in Seaborn before moving into advanced data analysis, dashboarding, and machine learning visualization projects.
