# OOP Basics in Python

This folder contains Python programs based on the core concepts of Object-Oriented Programming (OOP) in Python. It covers classes, objects, constructors, instance variables, methods, inheritance, encapsulation, and other foundational OOP ideas.

Object-Oriented Programming is an important programming paradigm that helps organize code into reusable and structured components. In Python, OOP is widely used to model real-world entities and build scalable applications.

Topics covered in this folder:
- classes and objects
- constructors
- instance variables
- methods
- inheritance
- method overriding
- encapsulation
- polymorphism basics
- class variables
- static methods

This folder is designed to build a strong foundation in OOP before moving to more advanced object-oriented programming concepts.
